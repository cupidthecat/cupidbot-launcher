How To Use CupidBot

Normal Account

Select a client version and profile, then use Play Normal Account.

Jagex Account

Use Login Jagex Account to connect, Add accounts for more characters, and Refresh to update names.

Profiles

Pick the RuneLite profile before launching. Jagex account profile choices are saved per character.

Client Version

Install local cupidbot-<version>.jar files into ~/.cupidbot, then choose the version here.

Client RAM

Increase RAM for larger plugin setups or lower it for lighter accounts.

SOCKS proxy

Enter ip:port:user:password only when the client should launch through a SOCKS proxy.

Plugins

Install hub plugin jars and plugins.json into ~/.runelite/cupidbot-plugins.

Folders and Logs

Open the menu button to jump to CupidBot folders, RuneLite folders, and launcher or client logs.
